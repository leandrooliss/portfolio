# Portfólio Pessoal — Leandro Oliveira

Este é o meu portfólio, desenvolvido para apresentar meus projetos, habilidades e um pouco sobre minha trajetória como desenvolvedor front-end e web designer.

## Tecnologias Utilizadas

HTML

CSS

JavaScript

Figma

## Estrutura do Projeto

O projeto está dividido em páginas principais:

Home (index.html)

Sobre (sobre.html)

Projetos (projetos.html)

Os estilos estão organizados da seguinte forma:

style.css: arquivo principal, com importação dos demais.

styles/: pasta contendo arquivos específicos (header.css, projetos.css, footer.css, etc.)

## Objetivo

Construir um site simples e moderno para expor meus trabalhos e experiências como desenvolvedor front-end e web designer.
