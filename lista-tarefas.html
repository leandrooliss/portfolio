Lista de Tarefas

A Lista de Tarefas foi um projeto desenvolvido com o objetivo principal de estudar JavaScript e aprofundar o entendimento sobre manipulação do DOM. A proposta era criar uma aplicação simples e funcional, que permitisse testar diferentes interações e dinâmicas entre elementos da interface de forma prática e direta.
O usuário pode adicionar novas tarefas, marcá-las como concluídas, editá-las e exclui-las, além de visualizar o horário em que cada item foi criado ou alterado. O design é simples e minimalista.
Esse projeto foi uma ótima oportunidade para consolidar conceitos importantes de JavaScript, como eventos, criação e remoção dinâmica de elementos, manipulação de classes e organização da lógica da aplicação. Também ajudou a aprimorar minha forma de estruturar layouts simples, mas funcionais, utilizando HTML, CSS e boas práticas de desenvolvimento.
